#### How to Initialize the Project.

- Clone the project.
    <pre>
    git clone https://github.com/count-on-mee/backend
    </pre>

- Install the node.js packages.
    <pre>
    cd backend
    npm install
    </pre>

#### How to Deploy the Server APIs

- Use the nodemon to execute the server.
    <pre>
    nodemon index.js
    </pre>
